## Overview
It is an fully Open Source Cyber Deck called **Open Deck** Which I have made from scratch it uses an Raspberry Pi 5 Model 2GB RAM as the main compute module and I have made an custom low profile keyboard for this project so that it will perfectly match with the Open Deck . And it also has an pico camera as the webcam .

## Why I chose this project?
Actually I am building an rocket launching system pad and for that I need to track gps , weather , Wind speed , direction a lot on the ground and for that I need something portable that I can use for it and which is seperate from my laptop so that it can run this task independently . Also I am huge fan of this hacking kind of hardware and an cyber deck is something like that so for that reason I have built it ..

##  How to use this project

This project can be used in multiple ways, depending upon the user need  like we can perform basic task like web browsing to fully automate our home to automate our 3-D printers also running some local AI agents into it. There are many possibilities through which we can use this project.For me I will use it to make some projects to be automated and also use in the rocket launching pad for those tasks .


## Features

* It uses custom low profile keyboard 
* It has web cam (pico cam )
* Runs on battery as well an power
* Uses Raspberry Pi 5 Model 2GB RAM
* Compact and Portable 
* Can run local models
* Custom 3d Printed Enclosure
* Fully Open Source
